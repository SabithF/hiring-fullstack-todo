This is the frontend of the Fullstack Todo App, built with **React + Vite** and styled using **TailwindCSS**.  
It communicates with the backend REST API to manage tasks.

---

##  Requirements

- Node.js 18+
- npm or yarn

---

##  Setup Instructions
- Create a .env file inside the client folder 
- Past below variables 

VITE_API_BASE_URL="http://localhost:5000/api/todos"

- Follow below instructions



### 1- Install Dependencies

```bash
npm install

```

### 1- Run the Development server
```bash
npm run dev

```

### The App will be available at http://localhost:5173/
